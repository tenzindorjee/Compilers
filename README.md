## Compilers Project

a rat18f compiler built in c++ for CPSC 323

### Prerequisites
any computer that can compile and execute c++ 

### Running the tests
run through terminal using 
```
g++ main.cpp 
./a.out
```
while in the project folder
might need to use 
```
g++ -std=c++11 main.cpp 
```
if using older versions of c++
